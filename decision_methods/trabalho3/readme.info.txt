O ficheiro locker_trab3 executa a simulação. Para isto, utilizar o compilador de python da versão 3:
	python3 locker_trab3

Posto isto, são criados dois ficheiros com os resultados:
	- simulation.sol : Todos os dados da simulação sobre a forma de uma tabela;
	- stats.sol : Os intervalos de confiança dos valores de custo e de máximo de pacotes para cada compensação.
